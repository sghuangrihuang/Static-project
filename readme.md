# 纯静态项目

## 为何创建这个静态页面仓库

在前端领域中也差不多混了一年了，回想当初为什么选择了前端，主要是大学的时候开始接触编程，那时候学过C，C#，,mssql，php，Objective-C，学校就是学着杂，而且自己没那种自学能力，课程一上完，写下作业就没了，后来有次机会让我接触到了前端，这些都是在16年写的一些静态页面，当时想写的是 与前后端完全分离，可事实公司就是说白的我充当一个切图仔的身份。做完页面直接扔给后台。
